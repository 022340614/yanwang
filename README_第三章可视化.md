# 第三章数据可视化网页

基于《数据可视化》第三章内容创建的交互式数据可视化展示页面。

## 项目概述

本网页展示了第三章中涵盖的各种Matplotlib图表配置和美化技巧，包括：

- 坐标轴标签和刻度设置
- 标题和图例添加
- 网格线显示
- 参考线和参考区域
- 注释文本添加
- 表格集成

## 包含的图表类型

1. **正弦曲线和余弦曲线** - 展示基础图表配置
2. **2019年内地电影票房排行榜** - 水平条形图
3. **支付宝月账单报告** - 饼图展示消费比例
4. **汽车速度与制动距离关系** - 散点图
5. **高二年级英语成绩评估** - 分组条形图
6. **阿里巴巴GMV增长趋势** - 柱状图

## 技术栈

- HTML5 + CSS3
- Chart.js (数据可视化库)
- 响应式设计

## 运行方式

### 电脑访问
1. 确保Python环境已安装
2. 在项目目录下运行：`python -m http.server 8000`
3. 打开浏览器访问：`http://localhost:8000/chapter3_visualization.html`

### 手机访问
1. 确保手机和电脑连接在**同一个WiFi网络**下
2. 在手机浏览器中输入：`http://10.212.21.122:8000/chapter3_visualization.html`
3. 如果无法访问，请检查防火墙设置或使用管理员权限运行服务器

## 特色功能

- **响应式设计**：完美适配桌面和移动设备
- **移动端优化**：针对手机和平板设备特别优化
- **交互式图表**：支持鼠标悬停和触摸操作
- **现代化UI**：渐变背景和卡片式布局
- **中文支持**：完整的中文字体配置
- **实时响应**：窗口大小变化时自动调整布局

## 移动端优化特性

### 布局优化
- 网格布局自动调整为单列显示
- 卡片间距和边距适配小屏幕
- 字体大小根据屏幕尺寸动态调整

### 图表优化
- 图例位置在移动端自动调整到底部
- 图表字体大小适配移动设备
- 触摸友好的交互体验
- 点半径和悬停效果优化

### 响应式处理
- 窗口大小变化时自动重新配置图表
- 支持横屏和竖屏切换
- 触摸设备特殊样式处理

这个可视化网页完美展示了第三章的教学内容，让用户能够直观地理解各种图表配置技巧。
