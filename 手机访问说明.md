# 手机访问数据可视化网页说明

## 问题原因
`localhost` 只能在本地电脑访问，手机无法通过 `localhost` 访问电脑上的服务器。

## 解决方案

### 方法1：使用电脑IP地址访问
1. 确保手机和电脑连接在**同一个WiFi网络**下
2. 在手机浏览器中输入：`http://10.212.21.122:8000/chapter3_visualization.html`
3. 即可正常访问数据可视化网页

### 方法2：使用二维码扫描（推荐）
1. 用手机扫描下面的二维码：
   ![二维码](https://api.qrserver.com/v1/create-qr-code/?size=200x200&data=http://10.212.21.122:8000/chapter3_visualization.html)

### 方法3：使用ngrok等工具（如果需要公网访问）
如果需要在外网访问，可以使用ngrok等工具：
```bash
ngrok http 8000
```

## 访问地址汇总
- **电脑本地访问**：`http://localhost:8000/chapter3_visualization.html`
- **手机访问**：`http://10.212.21.122:8000/chapter3_visualization.html`

## 注意事项
1. 确保电脑的防火墙允许8000端口的访问
2. 确保手机和电脑在同一个网络下
3. 如果IP地址发生变化，需要重新获取当前IP地址

## 获取当前IP地址的方法
在电脑上运行以下命令查看当前IP：
```bash
ipconfig | findstr "IPv4"
```

现在手机应该可以正常访问数据可视化网页了！
