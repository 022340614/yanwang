## 功能特性

### 数据可视化图表
1. **图书采购情况对比图** - 柱状图展示不同地区图书采购情况
2. **汇率走势图** - 折线图对比2017年与2019年汇率变化
3. **产品销售额趋势图** - 多产品季度销售数据可视化
4. **气温走势预测图** - 最高最低温度变化趋势

### 交互功能
- **科赫雪花可视化** - 可调节阶数和颜色的分形雪花
- **响应式设计** - 适配各种屏幕尺寸
